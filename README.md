# Akshaya Management System

A Django web application for managing customers, services, billing, and reports in an Akshaya / Internet Cafe center.

## Features
- Customer management
- Service management
- Multi-service billing
- Invoice generation
- PDF invoice download
- Excel report export
- Daily and monthly reports
- Dashboard analytics
- Staff login tracking

## Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
- Chart.js
- ReportLab
- OpenPyXL

## Project Link
[GitHub Repository](https://github.com/shanu313/akshaya-management-system)

## Use Case
This system is useful for Akshaya centers, internet cafes, and digital service shops to manage billing and customer records efficiently.
